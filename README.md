# AutoElite

AutoElite is a car selling landing page converted from a static HTML/CSS site into a React + Vite + Tailwind CSS project.

The current version keeps the same visual structure, color codes, and overall behavior from the original design while organizing the UI into reusable React components.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Project Structure

```text
src/
  components/
  data/
  pages/
  App.jsx
  main.jsx
```

- `node_modules/` and `dist/` are ignored through `.gitignore`
